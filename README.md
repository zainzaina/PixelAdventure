About

Pixel Adventure is a 2D platformer game developed using Unity. The project focuses on classic platforming gameplay combined with pixel art graphics, smooth animations, responsive controls, and engaging level design.

The game was created as part of my learning journey in Unity game development and demonstrates skills in programming, animation, game mechanics, UI design, and level creation.

Features
Pixel Art graphics
Smooth character animations
Enemy AI
Coins and collectibles
Health system
Multiple levels
Checkpoints
Sound effects
Responsive player controls
Clean user interface
Technologies
Unity
C#
Visual Studio
Pixel Art Assets
Animator Controller
Tilemap System
Gameplay

The player explores different levels while avoiding enemies, collecting coins, overcoming obstacles, and reaching the finish point.
